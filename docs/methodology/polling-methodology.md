# Polling Methodology

Last updated: 2026-06-30

## Purpose

Polling should help users understand movement in public opinion without overstating certainty. Plain Politics should show polling trends with source metadata, uncertainty, and clear separation between evidence and prediction.

## Core Rules

1. Show fieldwork dates prominently.
2. Separate individual polls from rolling averages.
3. Do not describe polling movement as an election prediction.
4. Do not overstate small movements that may be sampling noise.
5. Show source links and full tables where available.
6. Label stale polling data clearly.
7. Explain methodology changes in a public changelog.

## Required Poll Metadata

Store and display, where available:

1. Pollster
2. Client or commissioner
3. Fieldwork start date
4. Fieldwork end date
5. Publication date
6. Retrieval date
7. Sample size
8. Population sampled
9. Geography
10. Collection method
11. Question wording
12. Weighting notes
13. Full tables URL
14. Source URL
15. Source publisher

## Display Requirements

Polling views should include:

1. Individual poll table.
2. Trend or rolling-average chart.
3. Accessible table equivalent for charts.
4. Last updated timestamp.
5. Fieldwork date range.
6. Pollster and sample size.
7. Caveat that polling estimates are uncertain.
8. Methodology link.

## Movement Labels

Movement labels must be based on documented thresholds. Until thresholds are defined and tested, use cautious language:

1. `up in this poll`
2. `down in this poll`
3. `broadly unchanged`
4. `trend improving`
5. `trend weakening`

Avoid unsupported phrases such as `surging`, `collapsing`, or `set to win`.

## Averages

Before publishing rolling averages, define:

1. Included pollsters.
2. Recency window.
3. Weighting method.
4. Treatment of overlapping fieldwork.
5. Treatment of methodological outliers.
6. Minimum number of polls required.
7. How uncertainty is communicated.

## Out Of Scope Until Reviewed

1. Seat projections.
2. Tactical voting recommendations.
3. Election outcome prediction.
4. Automated explanations that infer cause without supporting evidence.
5. Demographic subgroup analysis without strong sample-size warnings.

## Open Decisions

1. Initial polling source list.
2. Rolling-average method.
3. Movement thresholds.
4. Pollster inclusion criteria.
5. Freshness thresholds for polling pages.
