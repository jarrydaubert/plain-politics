# Methodology

This directory is the operating source of truth for how UK Policy Explainer evaluates, stores, cites, and explains public political information.

## Standards

- `source-trust-framework.md` defines source tiers, inclusion rules, and blocked-by-default source types.
- `citation-policy.md` defines source-reference requirements for factual records, summaries, charts, and tables.
- `neutrality-standard.md` defines the operational neutrality checks for product copy, summaries, charts, and comparisons.
- `privacy-baseline.md` defines privacy-safe defaults for political data and user interactions.
- `polling-methodology.md` defines polling metadata, movement language, and uncertainty rules.

## Working Rules

1. No factual claim should ship without a source path.
2. Prefer primary public sources whenever they exist.
3. Record provenance at ingestion time, before rendering public pages.
4. Treat missing evidence as a product state: show uncertainty or coverage gaps.
5. Keep methodology changes dated and explain why they changed.
