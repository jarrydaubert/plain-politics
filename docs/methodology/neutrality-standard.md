# Neutrality Standard

## Purpose

Neutrality is an operating standard for evidence presentation. It applies to source ingestion, product copy, plain-English summaries, charts, maps, comparison layouts, and editorial review.

## Rules

1. Use the same structure across parties, politicians, policies, and donor records.
2. Do not infer motive, intent, dishonesty, or strategy without direct evidence.
3. Separate sourced facts from interpretation and uncertainty.
4. Use plain, non-persuasive language.
5. Surface disagreement between sources instead of smoothing it away.
6. Avoid ranking parties or policies unless the ranking method is public, deterministic, and cited.
7. Show coverage gaps consistently, including when a gap weakens a neat comparison.

## Prohibited Patterns

- Campaign-style calls to action
- Loaded adjectives for parties, politicians, donors, or voters
- Uneven scrutiny across comparable entities
- Selective citation that hides stronger contrary evidence
- Product experiences that advise tactical voting, persuasion, or campaign targeting

## QA Checklist

Use this checklist for product copy, content templates, charts, maps, and summaries:

1. Are all compared entities presented with the same fields?
2. Are unsupported claims removed or marked uncertain?
3. Are source limitations visible to users?
4. Would the wording still feel fair if applied to a different party?
5. Are refusal and fallback messages neutral and non-judgemental?
