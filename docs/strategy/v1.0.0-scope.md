# v1.0.0 Barebones Public Launch

Last updated: 2026-07-01

## Purpose

v1.0.0 is the smallest public version that proves the site is real, useful, neutral, and safe to build on.

It is not the full policy-comparison product. It should not try to cover a first policy area, ingest manifestos, track polling, analyse donations, or publish alignment scoring.

The v1.0.0 promise:

```text
I know little or nothing about UK politics. Help me start with my area, explain the words, show me live public records, and show me the sources.
```

## Must Ship

1. Public Vercel deployment with Cloudflare DNS/domain configured.
2. Cloudflare Email Routing aliases for contact and corrections, at minimum.
3. Beginner-first home page with clear `Start here`, `Find my MP`, `Glossary`, `Parliament`, `Sources`, and `About` routes.
4. Postcode-to-constituency/current-MP starter.
5. MP starter output: constituency, current MP, party, Commons membership start date, and source links.
6. Recent public parliamentary activity where reliable endpoints support it, labelled carefully as public record activity rather than local impact.
7. UK date/time in the navbar.
8. Political glossary page with sourced terms and traditions.
9. Parliament page with official live hooks for Commons party state, current members sample, upcoming business, and recent divisions.
10. Sources page showing hooked and candidate source families.
11. About page explaining the site, its no-prediction/no-advice limits, freshness, sources, and corrections without making the homepage about methodology.
12. Basic source failure, empty, loading, stale, and unavailable states for live data.
13. No demo data presented as real public data.
14. Build, unit tests, E2E smoke, and accessibility-conscious navigation passing in CI or documented release checks.

## Nice To Have, Not Blocking

1. Lightweight source health endpoint.
2. Cloudflare-checked health email to verified admin addresses.
3. First evergreen civic explainer, such as State Opening or PMQs.
4. Inline glossary links from the postcode and Parliament pages.
5. A public corrections page rather than only a corrections email alias.

## Explicitly Out Of v1.0.0

1. First policy area.
2. Party policy comparison.
3. Manifesto ingestion.
4. Polling tracker or polling averages.
5. Donations and political finance.
6. Alignment quiz or party-match scoring.
7. User accounts.
8. Change feed and RSS.
9. Public API and bulk downloads.
10. Seat projections or election prediction.
11. Any automated claim that an MP or party delivered, failed, broke a promise, or acted hypocritically.

## Feature Branch After v1.0.0

The first policy area should live on a feature branch after v1.0.0 is public.

Candidate branch:

```text
feature/first-policy-area
```

The branch should choose one issue, ingest official party and Parliament sources, map source excerpts, and produce the first Compare page only after source provenance is working for the barebones site.

## Launch Readiness Test

v1.0.0 is ready when a politics beginner can:

1. Visit the live URL.
2. Understand what the site is for within ten seconds.
3. Enter a postcode without creating an account.
4. See their constituency and current MP.
5. Understand at least one recent public record with glossary help.
6. Open or follow source links for factual claims.
7. Find the correction/contact route.
8. Leave without the site storing political views or raw postcode data by default.
